LAS VERSIONES HTML ventajas y desventajas

HTML

VENTAJAS: El HTML es idóneo para el aprendizaje por su marcado limpio y consistente.

Desventajas: El HTML es estático, o lo que es lo mismo, es un lenguaje pensado para páginas web estáticas o no dinámicas. Así mismo, por ser estático, no permite el manejo de base de datos.

HTML 2

VENTAJA: Permite describir hipertexto, tiene un despliegue rápido, lo reconoce y admite cualquier tipo de explorador y permite archivos pequeños.

DESVENTAJA: El diseño es más lento, tiene un lenguaje estático y las etiquetas son limitadas.

HTML 5

VENTAJA:es que es gratuito. No se requiere de ningún programa especial para programar en HTML5. Además, incorpora un código más ordenado para que el desarrollador pueda trabajar de forma más efectiva y detectar errores más rápidamente. • Es un lenguaje estático. • El proceso de diseño suele ser lento. • Su formato de vídeo es predeterminado.

DESVENTAJA: • Una de las principales desventajas de HTML5 para la creación de web es que ofrece un rendimiento aún bajo si lo comparas con las aplicaciones nativas. De hecho, la disponibilidad en cuanto a la creación de contenidos interactivos sigue siendo limitada. Se trata, aún, de un lenguaje estático.
